===== Version 2.0.0 finale =====
August 14, 2010, 15:22

Hop ! voici donc la 2.0.0 finale.
L’architecture est maintenant globalement stabilisée, la prochaine version se focalisera sur le jeu en réseau et le retour des scipts d’IA !
Le jeu est disponible sur Launchpad pour Ubuntu ou sur GitHub pour Windows et les autres distributions.

===== Nouveautés Alpha 2 =====
May 30, 2010, 00:20

La version 2.0.0 Alpha 2 devrait être prête dans les jours qui viennent. Quelques nouveautés intéressantes sont au programme :

  * Refonte des menus et des graphismes du tapis
  * Nouveaux Docks (statistiques, cartes jouées à tous les tours)
  * Donnes numérotées et manuelles
  * Editeur de donne

Il y a bien entendu de nombreux autres changements invisibles mais essentiels pour les futures évolutions du jeu.
N’oubliez pas d’aller voir la page du projet sur GitHub pour en apprendre plus. Sinon, tout est sur le dépôt Git !

Edit : j’ai mis sur Launchpad la release 2.0.0a2 empaquetée dans un beau paquet .deb. Disponible ici : https://launchpad.net/tarotclub
Edit2: la version Windows est disponible sur GitHub, packagée avec InnoSetup cette fois-ci.

===== Bienvenue ! =====
February 16, 2010, 18:54

Ce petit site centralise les activités autour du développement du jeu TarotClub, un jeu de Tarot libre (code source included et tout).

En plus, grâce à la librairie Qt, le jeu est disponible sur (potentiellement) tous les systèmes d’exploitation. Le rulez intégral quoi.
